MY NAME IS SITIAN CHEN

I am an enthusiastic developer of both front-end and back-end. For client-side development, I mainly use HTML, CSS, JavaScript and Bootstrap as front-end programming language. For server-side development, I mainly use Java SE, Java EE, EJB, Python and PHP as back-end programming language. For developing projects, such as management system and e-commerce platform, I mainly utilize JavaServer Faces as framework, PostgresSQL as database system, Wildfly(Jboss) as server, AWS or DigtalOcean as VPS and Ubuntu 18.04 as operating system. For doing data analytics, such as geovisualization, I mainly utilize QGIS, PostGIS, PostgresSQL and Ubuntu 18.04. Furthermore, due to my design background, I focus more on user experience, which is necessary for designing user interfaces.
